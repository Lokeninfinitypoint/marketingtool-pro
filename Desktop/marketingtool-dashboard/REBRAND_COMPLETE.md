## 💡 Pro Tips

### Customizing Colors
Edit `src/app/globals.css` to change colors:
```css
:root {
  --primary: #515FBC;        /* Change main color */
  --primary-dark: #322996;   /* Change dark variant */
  --primary-light: #917aff;  /* Change light variant */
}
```

### Using Gradient Classes
```tsx
<h1 className="text-gradient">Gradient Text</h1>
<div className="bg-gradient-primary">Gradient Background</div>
<div className="card-glow">Card with glow</div>
<div className="glass">Glassmorphism effect</div>
```

### Logo Variants
```tsx
import Logo from '@/components/Logo';

<Logo variant="full" size="md" />    // Full logo
<Logo variant="icon" size="lg" />    // Icon only
<Logo variant="wordmark" size="sm" /> // Text only
```
