# Audit

## Audit

The Audit tool helps you analyse your PPC accounts and uncover opportunities to improve performance.

### What it does

It scans your campaigns for common issues like underperforming keywords, missing ad extensions, and budget constraints.

### How to use it

1. Navigate to the Audit tool in the Tools section.
2. Choose the account or campaign you want to audit.
3. Review the findings and apply the recommended fixes.
