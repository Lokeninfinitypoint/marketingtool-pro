# Google Ads & Website Auditing

### What it does

This overview page explains the auditing features in Waudit.sbs for Google Ads and your website. It summarizes how the integrated tools analyse your campaigns and site performance and highlights the benefits of using them together.

### How to use it

* Navigate to each tool using the Tools section in the left sidebar.
* Start with the **Audit tool** to scan your ad campaigns and website for issues like underperforming keywords, missing ad extensions, and budget constraints.
* Use the **Performance tool** to monitor metrics and identify trends in your campaigns.
* Leverage the **Budget tool** to manage and optimize budget allocations for maximum efficiency.
* Experiment with different ad variants using the **Ad testing tool** to improve results.
