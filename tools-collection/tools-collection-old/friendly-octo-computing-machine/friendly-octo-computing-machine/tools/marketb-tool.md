# Marketb tool

### What it does

The Marketb tool integrates third‑party marketing analytics into Waudit.sbs. It helps you gather additional insights about your advertising campaigns and website performance, offering advanced analytics and recommendations beyond the built‑in tools.

### How to use it

* Ensure you have connected your Marketb account or API credentials within Waudit.sbs.
* Navigate to the Marketb tool under the Tools section in the sidebar.
* Choose the account or campaign you want to analyze.
* Review the analytics dashboards and insights provided by Marketb to understand performance trends and opportunities.
* Combine these insights with the findings from the Audit, Performance, Budget and Ad testing tools for a comprehensive view.
