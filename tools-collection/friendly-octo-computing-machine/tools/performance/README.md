# Performance

### Performance overview

The Performance tool provides a central dashboard to track how your PPC campaigns are performing. You can quickly see key metrics like click ‑ through rate (CTR), conversions, cost per conversion, and return on ad spend. With this insight, you can identify trends and optimize your campaigns.

#### How to use it

1. Navigate to **Performance** in the Tools section.
2. Select the account or campaign you want to review.
3. Review the charts and tables to understand which keywords and ads are performing well and which need improvement.
4. Adjust your bids, budgets and targeting based on these insights.

#### Tips

* Use the date range filter to compare performance over time.
* Segment your data by device, location or audience to uncover deeper insights.
* Export reports for sharing with your team or clients.
